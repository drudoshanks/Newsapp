Detailed Description

An interface for listening to decoder events and requests.

The IAudioProcessorListener interface is an interface exposed by the AudioDecoder to allow IAudioProcessor objects to notify the app of events.

virtual void intrasonics::IAudioProcessorListener::onNTPRequired ( )
pure virtual

Request an NTP update.

Called when the audio processor wants to request an NTP update. When the NTP request has completed, the audio processor's UpdateNTPInfo method can be called with the updated NTP information in an NTPInfo object.

Note
In the normal SDK, this event does not actually generate any NTP information for the plug-in and should not be used.
