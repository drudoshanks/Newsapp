Detailed Description

Informs the framework of the required sample rate.

Audio Processor plugins may need to advise the framework of the audio sample rate that they require. To do so they should implement the ISampleRateAdviser interface. The AudioDecoder will ask all plugins implementing the ISampleRateAdviser interface for their required sample rate through the GetPreferredSampleRate call. The AudioDecoder queries the available sample rates and chooses an appropriate sample rate to best support the plugins. If an AudioDecoder cannot find a sample rate that works with all plugins, those that cannot be supported will not be run. If an AudioProcessor does not implement ISampleRateAdviser, it is assumed to require 8kHz.

The following conversions are currently supported (the top row of the table is the sample rate to convert to, the leftmost column is the sample rate to convert from);

8 kHz 11.025 kHz 12 kHz 16 kHz 22.05 kHz 24 kHz 32 kHz 44.1 kHz 48 kHz
