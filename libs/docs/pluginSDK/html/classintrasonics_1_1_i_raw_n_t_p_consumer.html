virtual const IProcessorEventParams* intrasonics::IRawNTPConsumer::PrepareData ( const RawNTPInfo *  rawNtpInfo)
pure virtual

Indicates to the Raw NTP Consumer that the AudioDecoder is expecting it to prepare any data it has collected for uploading to the server.

Once the Audio Processor has prepared the data it should either report back the prepared data synchronously in the return value or report back through the IAudioProcessorListener::onProcessorEvent() method to indicate what kind of data is ready for upload.

Parameters
rawNtpInfoupdated raw NTP info acquired just before the call.
Returns
the event params describing the result of preparing data. If the implementer chooses to prepare the data asynchronously, this can be null
