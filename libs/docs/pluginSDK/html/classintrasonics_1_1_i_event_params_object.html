Detailed Description

An interface for EventParamsObject.

The IEventParamsObject interface is the ancestor of EventParamsobject that can be passed up from Audio Processors in a DictionaryEventParams object. IEventParamsObject is implemented by the template class EventParamsObject which allows the developer to provide specific typed information.
