Detailed Description

template<class ParameterType>
class intrasonics::EventParamsObject< ParameterType >

Contains a single piece of typed data for use in a DictionaryEventParams object.

The EventParamsObject<ParameterType> template class allows developers to wrap primitive types into a form suitable for passing up to the app layer via the DictionaryEventParams object in the OnProcessorEvent callback. The class is templated on ParameterType. Although developers could wrap any class with an EventParamsObject, only certain primitive types are supported for passing up to the app interface;
